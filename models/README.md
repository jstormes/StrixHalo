# Models Directory

This directory stores GGUF model files. Models are not included in the repository due to their size.

## Download Models

```bash
# Q8 quantization (~34GB) - Higher quality
./download-model.sh

# Q4 quantization (~18GB) - Smaller, faster
./download-model-q4.sh
```

## Expected Files

- `Qwen3-Coder-30B-A3B-Instruct-1M-UD-Q8_K_XL.gguf` (~34GB)
- `Qwen3-Coder-30B-A3B-Instruct-1M-UD-Q4_K_XL.gguf` (~18GB)
